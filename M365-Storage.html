Charger les données depuis SharePoint
Le bouton Charger les données tente de lire les 3 CSV SharePoint configurés dans Paramétrage, puis injecte les données dans les onglets OneDrive, Exchange et Visio.
✓ 3 sources chargées d’un coup · ✓ Session M365 requise · ✓ Détail d’erreur par source
Sources SharePoint par défaut
Ces URLs sont préremplies par défaut, mais restent modifiables. Le bouton Charger les données lit les 3 sources d’un coup.
État des sources
Après un chargement, cette zone indique pour chaque source : succès, accès refusé, fichier introuvable, réponse HTML au lieu d’un CSV, ou contenu vide.
Rappel technique
- La page tente une lecture réelle du fichier CSV.
- Si la session M365 du navigateur n’autorise pas l’accès, la source échoue.
- Si SharePoint renvoie une page HTML au lieu du CSV, la source est refusée avec le motif correspondant.
- Les URLs doivent pointer vers le fichier CSV lui-même.
